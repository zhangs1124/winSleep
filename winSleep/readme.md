# 自動休眠控制器

這個應用程式可以監控電腦的鍵盤活動，當使用者在指定時間內沒有任何操作時，自動讓電腦進入休眠模式。

## 功能特點
- 監控鍵盤活動
- 可自定義休眠時間（支援分鐘和秒數設定）
- 系統托盤圖示，方便控制
- 即時顯示倒數時間
- 自動進入休眠模式
- 休眠前可取消
- 可暫停/繼續計時功能

## 使用方法
1. 啟動應用程式
2. 在系統托盤中設定休眠時間（預設30分鐘）
3. 程式會自動在背景運行，監控使用者活動
4. 可以隨時暫停/繼續計時功能
5. 當檢測到無活動時間達到設定值時，系統將自動進入休眠模式

## 注意事項
- 任何鍵盤輸入都會重置計時器
- 系統進入休眠模式前會顯示確認對話框，可以選擇取消
- 系統進入休眠模式後，需要新的鍵盤活動才會重新開始計時
- 暫停計時時不會進入休眠模式
